## Introduction
This is the source code of the website [Le Rétro Projecteur](https://leretroprojecteur.com/).

## Spin up local version of website
- On command line:
  - Navigate to folder
  - Run: `python -m http.server 8000`
- On browser: `http://localhost:8000/`
